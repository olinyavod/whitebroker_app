# Сравнение проектов White Broker

В репозитории представлены два приложения для White Broker, разработанные на разных технологиях:

## 1. Flutter версия (`whitebroker_app/`)

**Технология:** Flutter/Dart

**Особенности:**
- Единая кодовая база для всех платформ
- Высокая производительность благодаря компиляции в нативный код
- Богатая экосистема виджетов Material Design
- JavaScript консоль для отладки WebView
- Поддержка загрузки файлов и работы с камерой

**Поддерживаемые платформы:**
- Android
- iOS
- Web
- Windows
- Linux
- macOS

**Размер приложения:**
- Android APK: ~20-30 MB (с движком Flutter)

## 2. .NET MAUI версия (`whitebroker_maui/`)

**Технология:** .NET MAUI/C#

**Особенности:**
- Нативный подход с использованием .NET
- XAML для описания интерфейса
- Прямой доступ к платформенным API
- Интеграция с экосистемой .NET
- Меньший размер приложения

**Поддерживаемые платформы:**
- Android
- iOS
- Windows
- macOS

**Размер приложения:**
- Android APK: ~15-20 MB

## Общие возможности обоих приложений

✅ Открытие веб-интерфейса в WebView  
✅ Полноэкранный режим  
✅ Поддержка HTTP (небезопасных соединений)  
✅ Работа с камерой  
✅ Загрузка файлов  
✅ Обработка ошибок загрузки  
✅ Индикатор загрузки страницы  

## Когда использовать Flutter версию?

- Нужна поддержка Web и Linux
- Команда знакома с Dart
- Требуется максимальная кроссплатформенность
- Нужны расширенные инструменты отладки WebView

## Когда использовать .NET MAUI версию?

- Команда знакома с C# и .NET экосистемой
- Проект уже использует .NET на бэкенде
- Важен меньший размер приложения
- Нужен более нативный подход для Windows

## URL приложения

Оба приложения открывают один и тот же URL:
```
http://89.104.65.16/login
```

## Установка и запуск

### Flutter
```bash
cd whitebroker_app
flutter pub get
flutter run
```

### .NET MAUI
```bash
cd whitebroker_maui
dotnet restore
dotnet run
```

## Выбор версии

Обе версии функционально эквивалентны и открывают один и тот же веб-интерфейс. Выбор между ними зависит от:
- Предпочтений команды разработчиков
- Технологического стека проекта
- Требуемых платформ
- Размера приложения

**Рекомендация:** Если вы новый пользователь и не уверены, какую версию выбрать, рекомендуем Flutter версию благодаря её большей универсальности и поддержке всех платформ, включая Web.
